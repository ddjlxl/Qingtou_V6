## 一、实体关系概览

```
┌─────────┐     ┌─────────┐     ┌─────────┐
│  用户   │────<│  订单   │>────│  车辆   │
│ (User)  │     │ (Order) │     │(Vehicle)│
└─────────┘     └────┬────┘     └────┬────┘
                     │               │
                     │          ┌────┴────┐
                     │          │  司机   │
                     │          │(Driver) │
                     │          └─────────┘
                     │
                ┌────┴────┐
                │ 常用地址 │
                │(Address)│
                └─────────┘

┌───────────┐     ┌─────────┐     ┌─────────┐
│   仓库    │────<│  库位   │>────│ 集装箱  │
│(Warehouse)│     │ (Slot)  │     │(Container)│
└───────────┘     └─────────┘     └─────────┘

┌───────────┐     ┌─────────────┐
│   证照    │     │   操作日志   │
│(Certificate)│   │(OperationLog)│
└───────────┘     └─────────────┘
```

## 二、数据类型说明

| 类型名称 | 中文说明 | 用途 |
|---------|---------|------|
| 唯一标识符 | UUID | 全局唯一编号，作为主键 |
| 字符串 | VARCHAR(长度) | 文本，长度限制 |
| 长文本 | TEXT | 长文本内容 |
| 整数 | INT | 整数值 |
| 小数 | DECIMAL(总长,小数位) | 金额、重量等精确小数 |
| 日期时间 | DATETIME | 日期和时间 |
| 日期 | DATE | 仅日期 |
| 布尔值 | BOOLEAN | 是/否 |
| 枚举 | ENUM | 固定选项 |

## 三、实体详细设计

### 3.1 用户表（User）

**说明**：系统登录用户，包含管理员、调度员、司机三种角色

| 字段名 | 数据类型 | 允许空 | 默认值 | 中文说明 |
|--------|---------|--------|--------|----------|
| 编号 | 唯一标识符 | 否 | 自动生成 | 主键，用户唯一编号 |
| 用户名 | 字符串(50) | 否 | - | 登录用户名，不能重复 |
| 密码 | 字符串(255) | 否 | - | 加密后的密码 |
| 姓名 | 字符串(50) | 否 | - | 真实姓名 |
| 手机号 | 字符串(20) | 是 | - | 联系电话 |
| 头像 | 字符串(255) | 是 | - | 头像图片地址 |
| 角色 | 枚举 | 否 | - | 角色：管理员/调度员/司机 |
| 状态 | 枚举 | 否 | 启用 | 状态：启用/禁用 |
| 创建时间 | 日期时间 | 否 | 当前时间 | 账号创建时间 |
| 更新时间 | 日期时间 | 否 | 当前时间 | 最后修改时间 |
| 最后登录时间 | 日期时间 | 是 | - | 最近一次登录时间 |

**角色选项**：
- 管理员：系统管理权限
- 调度员：创建任务、分配司机
- 司机：接单、更新状态

**状态选项**：
- 启用：可以正常登录
- 禁用：无法登录

**索引（加速查询）**：
- 主键索引：编号
- 唯一索引：用户名
- 普通索引：角色、状态

### 3.2 订单表（Order）⭐ 核心实体

**说明**：运输任务订单，V6将状态从9种简化为5种

| 字段名 | 数据类型 | 允许空 | 默认值 | 中文说明 |
|--------|---------|--------|--------|----------|
| 编号 | 唯一标识符 | 否 | 自动生成 | 主键 |
| 订单编号 | 字符串(50) | 否 | - | 显示编号，如：ORD20260503001 |
| 状态 | 枚举 | 否 | 待分配 | 订单状态 |
| 优先级 | 枚举 | 否 | 普通 | 优先级：普通/紧急 |
| 客户名称 | 字符串(100) | 否 | - | 货主/仓库方名称 |
| 客户电话 | 字符串(20) | 是 | - | 客户联系电话 |
| 起运地名称 | 字符串(200) | 否 | - | 货物出发地名称 |
| 起运地地址 | 字符串(500) | 否 | - | 起运地详细地址 |
| 目的地名称 | 字符串(200) | 否 | - | 货物到达地名称 |
| 目的地地址 | 字符串(500) | 否 | - | 目的地详细地址 |
| 箱号 | 字符串(20) | 是 | - | 集装箱号码 |
| 箱型 | 字符串(10) | 是 | - | 集装箱规格 |
| 封号 | 字符串(20) | 是 | - | 集装箱封号 |
| 司机编号 | 唯一标识符 | 是 | - | 分配的司机 |
| 车辆编号 | 唯一标识符 | 是 | - | 分配的车辆 |
| 调度员编号 | 唯一标识符 | 否 | - | 创建任务的调度员 |
| 备注 | 长文本 | 是 | - | 额外说明 |
| 识别图片 | 字符串(255) | 是 | - | OCR箱号识别照片地址 |
| 分配时间 | 日期时间 | 是 | - | 分配给司机的时间 |
| 开始时间 | 日期时间 | 是 | - | 司机开始运输的时间 |
| 完成时间 | 日期时间 | 是 | - | 任务完成时间 |
| 创建时间 | 日期时间 | 否 | 当前时间 | 订单创建时间 |
| 更新时间 | 日期时间 | 否 | 当前时间 | 最后修改时间 |

**状态选项（5种）**：
- 待分配：刚创建，未分配司机
- 已分配：已指定司机和车辆
- 运输中：司机已开始运输
- 已完成：任务完成
- 已超时：超过规定时间未完成

**箱型选项**：
- 20GP：20英尺普通箱
- 40GP：40英尺普通箱
- 40HQ：40英尺高箱
- 45HQ：45英尺高箱

**索引**：
- 主键索引：编号
- 唯一索引：订单编号
- 普通索引：状态、司机编号、车辆编号、调度员编号、创建时间

## 四、V4到V6的变更说明

### 4.1 删除的内容

| V4内容 | V6处理 | 原因 |
|--------|--------|------|
| 车队实体 | 删除 | 简化管理 |
| 路线模板 | 删除 | 用常用地址替代 |
| 订单9种状态 | 简化为5种 | 减少混淆 |
| 车辆4种状态 | 简化为3种 | 删除"异常"状态 |
| 仓库管理员角色 | 删除 | 由调度员兼任 |
| 订单锁定功能 | 删除 | 简化流程 |
| EIR编号 | 删除 | 简化字段 |
| 途经点 | 删除 | 简化路线 |
| 单据附件 | 删除 | 简化流程 |
| 电子邮箱 | 删除 | 不需要 |
| 起运地联系人/电话 | 删除 | 不需要 |
| 目的地联系人/电话 | 删除 | 不需要 |
| 业务类型（进口/出口/转运） | 删除 | 不需要 |
| 预估重量 | 删除 | 不需要 |
| 实际重量 | 删除 | 不需要 |
| 运输费用 | 删除 | 不需要 |

### 4.2 新增的内容

| 新增内容 | 说明 |
|---------|------|
| 司机独立表 | 极简设计，只保留核心字段 |
| 仓库客户名称 | 与订单客户名称一致，方便出库创建任务 |
| 外协车辆管理 | 简化录入，关联司机表记录司机信息 |
| 帮助中心 | 后台可编辑的操作手册 |

### 4.3 修改的内容

| 修改内容 | 说明 |
|---------|------|
| 归属性质 | 自有/外协（原：自有/挂靠） |

| 司机 | 绑定车辆编号 | 车辆 | 可选关联 |

| 操作日志 | 操作人编号 | 用户 | 必须关联 |

### 6.2 级联规则

| 父表操作 | 子表处理 | 说明 |
|---------|---------|------|
| 删除用户 | 禁止删除 | 有关联订单/司机/日志时不能删除 |
| 删除司机 | 禁止删除 | 有关联订单/车辆时不能删除 |
| 删除车辆 | 禁止删除 | 有关联订单时不能删除 |
| 删除仓库 | 级联删除库位 | 删除仓库同时删除所有库位 |
| 删除订单 | 保留记录 | 订单为历史数据，不允许物理删除 |

### 6.3 软删除策略

| 实体 | 策略 | 说明 |
|------|------|------|
| 用户 | 状态标记 | 禁用状态视为删除 |
| 司机 | 状态标记 | 禁用状态视为删除 |
| 车辆 | 状态标记 | 禁用状态视为删除 |
| 仓库 | 物理删除 | 无历史关联，可物理删除 |
| 库位 | 物理删除 | 随仓库级联删除 |
| 订单 | 禁止删除 | 业务数据，不允许删除 |
| 常用地址 | 物理删除 | 无历史关联 |
| 证照 | 物理删除 | 无历史关联 |
