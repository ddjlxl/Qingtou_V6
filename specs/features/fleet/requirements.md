## 功能概述
- **功能名称**：Fleet 车队管理
- **功能描述**：管理车辆和司机的档案、状态、证照，以及运输流水记录，为调度中心提供车辆和司机资源支持
- **目标用户**：调度员、管理员
- **优先级**：高（MVP 核心模块）

## 核心流程

**车辆管理流程**：调度员/管理员在"车辆管理"Tab 中新增车辆（填写车牌、归属性质），车辆状态默认为"空闲"。后续可通过编辑绑定司机。当调度员在调度中心分配任务给该车辆时，状态自动变为"运输中"；任务完成后恢复"空闲"；任务超时则变为"超时"。

**司机管理流程**：调度员/管理员在"司机管理"Tab 中新增司机（填写姓名、手机号），司机与车辆固定绑定。

**证照管理流程**：调度员/管理员在"证照管理"Tab 中为车辆或司机上传证照（选择类型、上传照片、填写有效期和备注）。系统自动检查证照有效期，提前30天在车队管理页的统计概览 Tab 显示预警。

**运输流水流程**：每月底，调度员在调度中心导出已完成任务，然后在"运输流水"Tab 中导入文件，系统自动统计每个司机/车辆的任务数量。

## 验收标准（AC）

### Happy Path（正常流程）

#### 车辆管理

- **AC-001**: Given 调度员在"车辆管理"Tab，When 填写车牌号、归属性质后点击"保存"，Then 车辆创建成功，出现在列表中，状态显示为"空闲"（绿色）。

- **AC-002**: Given 调度员在车辆列表中点击某车辆的"编辑"，When 修改归属性质或关联司机后点击"保存"，Then 车辆信息更新成功，列表显示最新信息。

- **AC-003**: Given 调度员在"车辆管理"Tab，When 查看车辆列表，Then 显示所有车辆的车牌、归属性质、关联司机、状态（颜色区分）。

- **AC-004**: Given 调度员在车辆列表页，When 点击状态筛选（空闲/运输中/超时），Then 列表只显示对应状态的车辆。

#### 司机管理

- **AC-005**: Given 调度员在"司机管理"Tab，When 填写姓名、手机号后点击"保存"，Then 司机创建成功，出现在列表中。

- **AC-006**: Given 调度员在司机列表中点击某司机的"编辑"，When 修改姓名或手机号后点击"保存"，Then 司机信息更新成功，列表显示最新信息。

- **AC-007**: Given 调度员在"司机管理"Tab，When 查看司机列表，Then 显示所有司机的姓名、手机号、关联车辆。

#### 证照管理

- **AC-008**: Given 调度员在"证照管理"Tab，When 选择证照类型、填写证照名称、签发日期、有效期至、上传照片（可选）、填写备注、选择所属对象（车辆或司机）后点击"保存"，Then 证照创建成功，出现在列表中。

- **AC-009**: Given 调度员在证照列表中点击某证照的"编辑"，When 修改有效期、备注或重新上传照片后点击"保存"，Then 证照信息更新成功。

- **AC-010**: Given 调度员在"证照管理"Tab，When 查看证照列表，Then 显示所有证照的类型、所属对象、有效期、状态（正常/即将到期）。

- **AC-011**: Given 调度员在证照列表页，When 点击"预警筛选"，Then 列表只显示30天内到期的证照。

- **AC-032**: Given 调度员在证照列表中点击某证照的"删除"，When 确认删除操作，Then 证照从列表中移除，数据库中删除该记录。

#### 运输流水

- **AC-012**: Given 调度员在"运输流水"Tab，When 点击"导入"按钮，选择 Excel 或 txt 文件（按模板格式），Then 系统解析文件，导入成功后显示"成功导入 N 条记录"。

- **AC-013**: Given 调度员在"运输流水"Tab，When 查看流水列表，Then 显示所有记录的任务编号、客户信息、起运地、目的地、箱号、执行车辆、执行司机。

- **AC-014**: Given 调度员在流水列表页，When 选择时间范围、车辆或司机进行筛选，Then 列表只显示符合条件的记录。

- **AC-015**: Given 调度员在"运输流水"Tab，When 查看统计区域，Then 显示每个司机/车辆的任务数量统计。

- **AC-033**: Given 调度员在运输流水列表页，When 记录数量超过一页，Then 列表自动分页，底部显示分页控件（上一页/下一页/页码）。

#### 车队统计

- **AC-016**: Given 调度员在"统计概览"Tab，When 查看证照预警卡片，Then 显示30天内到期的证照数量，点击可跳转到证照管理Tab。

- **AC-017**: Given 调度员在"统计概览"Tab，When 查看本月任务数卡片，Then 显示本月导入的运输流水记录总数。

### Edge & Error Cases（边界和异常）

#### 车辆管理

- **AC-018**: Given 调度员新增车辆时，When 输入的车牌号已存在，Then 系统拒绝创建，提示"车牌号已存在，请检查"。

- **AC-019**: Given 调度员尝试删除某车辆，When 该车辆有历史运输流水记录，Then 系统拒绝删除，提示"该车辆有历史记录，无法删除，可标记为停用"，并提供"停用"按钮。若无历史记录，则允许删除，同时级联删除该车辆的所有关联证照（含附件文件）。

- **AC-020**: Given 调度员在调度中心分配任务，When 选择的车辆状态为"运输中"，Then 系统拒绝分配，提示"该车辆正在运输中，请选择其他车辆"。

#### 司机管理

- **AC-021**: Given 调度员尝试删除某司机，When 该司机有历史运输流水记录，Then 系统拒绝删除，提示"该司机有历史记录，无法删除，可标记为停用"，并提供"停用"按钮。若无历史记录，则允许删除，同时级联删除该司机的所有关联证照（含附件文件）。

- **AC-034**: Given 调度员新增或编辑司机时，When 输入的手机号格式不正确（非 11 位数字或不以 1 开头），Then 系统拒绝保存，提示"手机号格式不正确"。

- **AC-035**: Given 调度员新增司机时，When 输入的手机号已被其他司机使用，Then 系统拒绝创建，提示"手机号已存在，请检查"。

#### 证照管理

- **AC-022**: Given 某证照将在30天内到期，When 调度员进入"统计概览"Tab，Then 证照预警卡片显示该证照数量，且数字加粗高亮。

- **AC-023**: Given 调度员上传证照照片，When 文件大小超过 5MB 或格式不支持，Then 系统提示"文件大小不能超过 5 MB"或"仅支持 JPG/PNG 格式"。

- **AC-036**: Given 系统级联删除车辆/司机关联证照时，When 证照的附件文件已被手动删除（文件不存在），Then 系统静默跳过文件删除，记录 warning 日志，不阻断数据库删除操作。

#### 运输流水

- **AC-024**: Given 调度员导入运输流水文件，When 文件格式不是 Excel 或 txt，Then 系统拒绝导入，提示"仅支持 Excel (.xlsx) 或 txt 格式"。

- **AC-025**: Given 调度员导入运输流水文件，When 文件列数与模板不匹配，Then 系统拒绝导入，提示"文件格式错误，请下载模板后重新填写"。

- **AC-026**: Given 调度员导入运输流水文件，When 文件中包含已存在的任务编号，Then 系统提示"发现 N 条重复记录，已自动跳过，成功导入 M 条新记录"。

### Business Rules（业务规则）

- **AC-027**: Given 调度员在调度中心分配任务给某车辆，When 任务分配成功，Then 该车辆状态自动从"空闲"变为"运输中"（蓝色）。

- **AC-028**: Given 某车辆正在执行任务，When 从任务开始时间起超过4小时仍未完成，Then 该车辆状态自动从"运输中"变为"超时"（红色）。

- **AC-029**: Given 某证照距离有效期不足30天，When 系统每日自动检查证照有效期，Then 该证照计入预警数量，显示在统计概览Tab。

- **AC-030**: Given 调度员新增车辆时选择关联司机，When 该司机已关联其他车辆，Then 系统提示"该司机已关联车辆 XXX，是否更换关联？"，确认后原车辆的关联司机字段清空。

- **AC-031**: Given 调度员或管理员登录系统，When 访问车队管理模块，Then 两者权限相同，均可对车辆、司机、证照、运输流水进行增删改查操作。

## 数据规格

### 车辆信息
- **车牌号**：必填，唯一，字符串
- **归属性质**：必填，枚举（自有车辆 / 外协车辆）
- **关联司机**：可选，关联司机表 ID
- **状态**：自动计算，枚举（空闲 / 运输中 / 超时）
- **超时阈值**：任务开始后超过 4 小时未完成，自动标记为"超时"
- **停用状态**：布尔值，默认 false

### 司机信息
- **姓名**：必填，字符串
- **手机号**：必填，唯一，字符串
- **停用状态**：布尔值，默认 false

### 证照信息
- **证照类型**：必填，枚举（行驶证 / 道路运输证 / 交强险 / 商业险 / 年审 / 驾驶证 / 从业资格证）
- **证照名称**：必填，字符串（如"粤A12345行驶证"、"张三驾驶证"）
- **签发日期**：必填，日期
- **有效期至**：必填，日期
- **证照照片**：可选，文件路径
- **备注**：可选，字符串
- **所属对象类型**：必填，枚举（车辆 / 司机）
- **所属对象 ID**：必填，关联车辆表或司机表 ID

### 运输流水记录
- **任务编号**：必填，字符串
- **客户信息**：必填，字符串
- **起运地**：必填，字符串
- **目的地**：必填，字符串
- **箱号**：必填，字符串
- **执行车辆**：必填，存储关联车辆表 ID；导入时填写**车牌号**（唯一标识），系统自动匹配转换为车辆 ID
- **执行司机**：必填，存储关联司机表 ID；导入时填写**手机号**（唯一标识），系统自动匹配转换为司机 ID
- **导入时间**：自动生成，日期时间
- **导入文件格式**：Excel (.xlsx) 或 txt（Tab 分隔）

## 页面结构

### 单页面 + 5个Tab布局

```
车队管理（路由：/fleet）
├── Tab 1: 统计概览
│   ├── 证照预警卡片（显示30天内到期数量，点击跳转证照管理Tab）
│   └── 本月任务数卡片（显示本月导入的流水记录总数）
├── Tab 2: 车辆管理
│   ├── 车辆列表表格（车牌、归属性质、关联司机、状态）
│   ├── 新增/编辑车辆弹窗
│   ├── 状态筛选（空闲/运输中/超时）
│   └── 停用按钮（有历史记录的车辆）
├── Tab 3: 司机管理
│   ├── 司机列表表格（姓名、手机号、关联车辆）
│   ├── 新增/编辑司机弹窗
│   └── 停用按钮（有历史记录的司机）
├── Tab 4: 证照管理
│   ├── 证照列表表格（类型、所属对象、有效期、状态）
│   ├── 新增/编辑证照弹窗（上传照片、填写有效期和备注）
│   ├── 预警筛选（30天内到期）
│   └── 按车辆/司机筛选
└── Tab 5: 运输流水
    ├── 流水列表表格（任务编号、客户信息、起运地、目的地、箱号、执行车辆、执行司机）
    ├── 时间范围筛选（本月、上月、自定义）
    ├── 车辆/司机筛选
    ├── 导入按钮（弹窗上传文件）
    ├── 模板下载链接
    └── 统计区域（每个司机/车辆的任务数量）
```

## 范围界定

- ✅ 运输流水统计（每个司机/车辆的任务数量）
- ✅ 车队统计卡片（证照预警数、本月任务数）
- ✅ 单页面 + 5个Tab布局
- ✅ 车辆状态冲突处理（运输中的车辆拒绝分配新任务）
- ✅ 有历史记录的车辆/司机不能删除，只能停用

### 本次不做
- ❌ 里程统计（今日里程、本月里程）
- ❌ 任务排队功能（车辆运输中时新任务排队等待）
- ❌ 证照历史变更记录
- ❌ 证照预览功能
- ❌ 运输流水导出功能（在调度中心导出，不在 fleet 模块）
- ❌ 智能匹配导入文件列名
- ❌ 停用后恢复功能（车辆/司机停用后重新启用）
