## 六、问题与解决

### 6.1 枚举导入问题

**问题**：ContainerType 和 BusinessType 作为类型导出，无法作为值使用

**解决方案**：直接从 order.ts 文件导入枚举值

```typescript
// ❌ 错误：从 dispatch 模块导入（类型导出）
import { ContainerType } from '@/modules/dispatch'

// ✅ 正确：从 order.ts 文件导入（枚举值）
import { ContainerType } from '@/modules/dispatch/types/order'
```

### 6.2 window.matchMedia 问题

**问题**：测试环境中 window.matchMedia 不存在

**解决方案**：在测试文件中 mock window.matchMedia

```typescript
Object.defineProperty(window, 'matchMedia', {
  writable: true,
  value: vi.fn().mockImplementation((query: string) => ({
    matches: false,
    media: query,
    onchange: null,
    addListener: vi.fn(),
    removeListener: vi.fn(),
    addEventListener: vi.fn(),
    removeEventListener: vi.fn(),
    dispatchEvent: vi.fn(),
  })),
})
```
