## 六、问题与解决

### 6.1 Element Plus 组件查找问题

**问题**：通过 `findAllComponents({ name: 'ElButton' })` 查找按钮时，Element Plus Dialog footer 插槽中按钮未被正确识别

**解决方案**：改用 `findAll('button')` 并通过文本内容查找按钮

### 6.2 Vue 3 ref 访问问题

**问题**：Vue 3 `<script setup>` 中 ref 默认不暴露，直接访问 `vm.customerName.value` 导致错误

**解决方案**：移除对内部 ref 的直接访问，仅测试公开接口（如 handleSubmit 方法）

### 6.3 组件测试策略

**策略**：通过直接调用组件方法（如 handleSubmit）而非模拟用户交互，提高测试稳定性
