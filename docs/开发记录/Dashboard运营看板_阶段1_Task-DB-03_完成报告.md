## 浏览器验收记录

- 页面左上角浮动 4 个指标卡片，半透明毛玻璃背景
- 今日任务数显示整数（0/15 均正确）
- 完成率显示百分比（0%/86% 均正确，0.856 → 86%）
- 超时数 > 0 时红色（rgb(245,108,108) = #f56c6c），= 0 时默认色
- 平均转运时间有值显示"XX分钟"，null 显示"--"
- pointer-events: 外层 none，内层 auto，不遮挡地图交互
- 1366×768 分辨率下不溢出（卡片 right: 644 < viewport: 1366）
