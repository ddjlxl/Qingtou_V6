## 六、问题与解决

### 6.1 Vue watch 监听问题

**问题**：mock 的 keyword 和 searchHighlights 需要是真正的 ref 对象，而不是普通对象。Vue 的 watch 无法监听普通对象。

**解决方案**：使用 `ref()` 创建真正的响应式对象

```typescript
// ❌ 错误：普通对象无法被 watch 监听
const mockKeyword = { value: '' }

// ✅ 正确：使用 ref 创建响应式对象
const mockKeyword = ref('')
```

### 6.2 事件触发问题

**问题**：`trigger('slot-click', loadedSlot)` 会把 loadedSlot 包装成 Event 对象，导致断言失败

**解决方案**：直接调用组件方法 `handleSlotClick` 而不是通过事件触发

```typescript
// ❌ 错误：trigger 会包装数据
await zoneCard.trigger('slot-click', loadedSlot)

// ✅ 正确：直接调用方法
const vm = wrapper.vm as unknown as { handleSlotClick: (slot: Slot) => void }
vm.handleSlotClick(loadedSlot)
```
