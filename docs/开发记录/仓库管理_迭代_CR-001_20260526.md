# 仓库管理迭代记录 — CR-001

## 基本信息
- **CR 编号**：CR-001
- **日期**：2026-05-26
- **变更类型**：范围内变更
- **触发原因**：用户反馈移动模式体验不佳，每次移动后需要重新点击"移动"按钮，效率低下

移动完成后保持移动模式，支持连续移动多个集装箱，提升批量操作效率。新增 ESC 键快捷退出功能。

async function move(sourceSlotId: string, targetSlotId: string) {
  await warehouseService.move(sourceSlotId, targetSlotId)

### 2. 键盘快捷键支持
**文件**：`WarehousePage.vue`

新增 ESC 键监听，在移动模式下按 ESC 键退出：
```typescript
function handleKeyDown(event: KeyboardEvent) {
  if (event.key === 'Escape' && store.isMoveMode) {
    store.toggleMoveMode()
  }
}

onMounted(() => {
  store.init()
  window.addEventListener('keydown', handleKeyDown)
})

onUnmounted(() => {
  window.removeEventListener('keydown', handleKeyDown)
})
```

## 用户体验改进

### 改进前
1. 用户点击"移动"按钮进入移动模式
2. 选择源库位 → 选择目标库位 → 移动完成
3. **自动退出移动模式**
4. 用户需要再次点击"移动"按钮才能继续移动

**问题**：批量整理仓库时需要重复点击"移动"按钮，效率低下

### 改进后
1. 用户点击"移动"按钮进入移动模式
2. 选择源库位 → 选择目标库位 → 移动完成
3. **保持移动模式，可继续移动其他集装箱**
4. 用户完成所有移动后，点击"取消移动"或按 ESC 键退出

**优势**：
- 符合"批量整理仓库"的真实场景
- 减少重复操作，提高效率
- ESC 键快捷退出，操作更便捷
