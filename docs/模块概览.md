## 模块依赖关系

```
                    ┌─────────┐
                    │   auth  │
                    └────┬────┘
                         │
         ┌───────────────┼───────────────┐
         │               │               │
    ┌────┴────┐    ┌─────┴─────┐   ┌─────┴─────┐
    │ dispatch│    │   fleet   │   │ warehouse │
    └────┬────┘    └─────┬─────┘   └─────┬─────┘
         │               │               │
         └───────────────┴───────────────┘
                         │
                    ┌────┴────┐
                    │  driver │
                    └─────────┘
```

- `auth`: 基础模块，被所有模块依赖
- `dispatch`: 调度订单，与 fleet 联动（派单时选择车辆/司机）
- `fleet`: 车队资源管理，为 dispatch 提供可用资源
- `warehouse`: 仓库管理，独立模块
- `driver`: 司机移动端，依赖 dispatch 和 fleet 的数据
